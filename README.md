# Stone Chalengge 
## Backend Developer with Node
---

### You will need 

- **Git**
- **Node**

---

### Clone repository

Cloning with protocol **HTTP**:

```sh
git clone https://
```

or

Cloning with protocol **SSH**:

```sh
git clone git@git
```

---


### Change directory 

```sh
cd stoneChallenge/
```

and after


### Install dependencies

With **NPM**:

```sh
npm install
```

**or**


With **Yarn**:

```sh
yarn
```

### Execute project in developer mode


```sh
npm run dev
```

**or**


With **Yarn**:

```sh
yarn dev
```


### Execute test


```sh
npm run test
```

**or**


With **Yarn**:

```sh
yarn test
```
